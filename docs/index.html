RVSC Viewer

Read-only viewer for Victron VEConfigure .rvsc settings files — decodes and displays, never writes.

Your file never leaves your computer — parsing happens entirely in your browser, using the FileReader API. This page makes no network requests: no upload, no analytics, no external fonts or scripts. You can verify this yourself by opening your browser's network inspector, or by disconnecting from the internet before loading a file.
Read-only tool. This viewer decodes and displays settings. It cannot save, modify, or export a changed .rvsc file, and will never offer to. To change device settings, use Victron's own VEConfigure or VRM Remote VEConfigure — see README and NOTICE.md.

RVSC Viewer is an independent, community project. Not affiliated with, endorsed by, or sponsored by Victron Energy B.V. "Victron", "VEConfigure" and related marks are used only to identify the file format this tool reads.
